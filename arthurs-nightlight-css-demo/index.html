[CSS] Arthur's Nightlight (one handed)

11 February 2019 on CSS

I originally created this in the few months after Arthur was born, during a period where I was simultaneously taking much needed parental leave, and learning about front-end code. Â I had originally wrote this blurb about my experience:

When I'm not working on my front-end skills, I'm likely spending time with my son, or just trying to get him to go to bed. I've spent quite a bit of time in the last few months in a dark room while I swaddle / shush / swing him to sleep. So often, in fact, I felt compelled to recreate his nightlight from memory.

I can't believe I can even remember coding this demo. Man, I can't. Â OK, I have an excuse because of the sleep depredation, but I do remember this part:

I did this a majority project with one hand.

A lot of the initial layout and composition was done by manually adapting a very rough SVG draft (I mean... rough) to an HTML struct.

Then... I did what nature does. Â I bent shape into a light and... voila.
How did I bend it exactly? Well... a lot of < ^ > and +- keys.

Through a combination of absolute positioning, well-timed %, and several view-port metrics, I remember being very excited at some of the responsive behaviors I was able to obtain out of this little .toy object.

There's definitely some issues when used on other browsers, and I fixed a few through a few more burbs and knee bounces. Â Overall, there's not a lot of practical uses for something like this, but I feel I learned a lot about the ways you can re-shape essential CSS elements, in addition to exploring newer browser APIs.
