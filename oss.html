FieldTest — TypeScript Validation Toolkit

Comprehensive validation for content management and schema-driven apps across modern frameworks.

GitHub Repo Examples README

  • Deterministic schemas and pluggable rules
  • Works in Node, Next.js, Astro, and CLIs
  • Great for migrations, docs freshness, and content quality gates

Products that use FieldTest

  • Docs Score — documentation health assessment
  • WorkTrace — QR-based field service management
  • CSV Processor — automated, validated data pipelines
  • RFP Radar — procurement monitoring

These live on Westmark.dev for contractor audiences; FieldTest itself remains open and independent here.

brickwave

Offline deterministic noise generator for DSP devs, AI researchers, and sound designers.

  • Generate white, pink, and brown noise
  • Seed-based deterministic generation for reproducible results
  • Stereo output support
  • Envelope shaping with fade-in/fade-out
  • Pacific Northwest ambient layers (rain, birds, running water)
  • Configurable duration and sample rate
  • High-quality 16-bit PCM WAV output
  • Minimal dependencies (only commander)
  • Fast TypeScript implementation
